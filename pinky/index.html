What is Pinky?

Pinky is a microcontroller board, similar to an Arduino, that I built and designed for use in my projects. It uses Microchip's PIC16F15243 as it's MCU. I then wrote a script and Arduino program in order to program the device. The final version of Pinky will have a bootloader and will support USB programming similar to Arduino. However, a key feature that Pinky has above of Arduino is the inclusion of a number of 4-pin JST connectors for use with SPI. This allows Pinky to be a host board for a number of connected client boards.

Pinky's features:

  • 3x 4-pin JST-PH connectors, with SPI
  • Programmable via Micro USB
  • 2 wire power input
  • 5-pin connector for low-voltage progrmming
  • Small form factor

Plans for Pinky v1.1

There are currently several flaws with this board, that will be addressed in Pinky v1.1:

  • The GPIO layout is terrible (doesn't really support shields)
  • CH340C needs a 4.7k pull down resistor on DTR (easy fix with THT resistor)
  • Green status LED is far too bright
  • Micro USB is old
  • Reset switch does nothing
  • Mounting with screws is limited
  • Some components are useless because of LVP

Future plans

Some future version of Pinky will have these features:

  • Some form of RF transceiver
  • 32-bit MCU
